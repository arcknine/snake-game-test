# Snake using Gosu GEM

* Ruby Version 2.5.0

How To Run:

1. `gem install gosu`

2. `ruby snake.rb` to run game

 
# Snake Running in Terminal

* Ruby Version 2.5.0

* Controls: `W (Up), A (Left), S (Down), D (Right), Q (to quit)`

How To Run:

1. `gem install curses`

2. `ruby snake_terminal.rb` to run game
